# Usage

```bash
# {content} is the content of the QR code
$ python qr.py {content}
```

# Installation

```bash
# Installing the dependencies
$ pip install qrcode

# cloning the repository on your cwd (current working directory)
$ git clone https://github.com/HrsSid/QrCodeGenerator
```
